## Goal

Implement the first two screens of the NASI onboarding flow — the **mode picker** and **app connect** screens. These establish the core design language (color-coded modes, mobile layout, rounded card system) that all subsequent screens inherit.

**Source**: FigJam board pulled via Figma MCP — `https://www.figma.com/board/O9ONqUiEu0vcaAJ5VGgq1d/NASI?node-id=39-825`

This is the proof-of-concept story: demonstrating that a FigJam lo-fi flow can be read via MCP and translated into working Astro + React components.

## Design Language Established Here

- **Loom** = blue
- **Sprite** = pink/coral
- **Shell** = gray
- Mobile-first (390px viewport target)
- Tall rounded rectangle cards as primary UI primitive
- Tab bar navigation shared across screens 2–6
