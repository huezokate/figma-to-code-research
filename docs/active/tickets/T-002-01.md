## Context

Implement **Screen 3** of the NASI onboarding flow: the Sprite glow-up transition moment.

**FigJam source**: `https://www.figma.com/board/O9ONqUiEu0vcaAJ5VGgq1d/NASI?node-id=39-825`

### What the FigJam shows

A transitional "loading" screen shown while NASI builds the user's profile from the connected apps.

**Large centered ellipse** (fills most of the screen) with the note:
> "craping moment when Sprite goes through a glow up"

Below the ellipse: an AI-generated character portrait image (a glowing anime-style character — placeholder for the actual Sprite avatar).

This is a **moment of delight** — the app is doing something (scraping/personalizing) and the UI makes it feel like Sprite is coming alive for you specifically.

### What to build

A React component `NasiGlowUp.tsx` in `experiments/003-nasi-glow-up/src/`. Focus on:
- Full-screen dark background
- Large pulsing circle/ellipse (CSS animation — pulse or glow effect)
- A placeholder image or gradient inside the circle representing the Sprite avatar
- Subtle loading text below (e.g., "Sprite is getting to know you…")
- Use mode colors from T-001-01 (Sprite = pink/coral as the glow color)

No tab bar on this screen — it's a full-screen interstitial.
